You have the following HTML segment:

<div id="main">
      <div id="sub">JavaScript</div>
      <div>TypeScript</div>
      <div>SQL</div>
      <div>C++</div>
      <div>Java</div>
      <div>Smalltalk</div>
</div>

You need to remove all the child divs.

Which code should you use?

let n = document.getElementById("main"); while (n.firstChild) {      n.removeChild(n.firstChild); }
let n = document.getElementById("sub"); if (n.parentNode)      n.parentNode.removeChild(n);
let main = document.getElementById("main"); let sub = document.getElementById("sub"); let something = main.removeChild(sub); while(true)      something = main.removeChild(sub);
let main = document.getElementById("main"); let sub = document.getElementById("sub"); let something = main.removeChild(sub);

Explanation

You should use the following JavaScript code the removes all the child divs:

let n = document.getElementById("main");
while (n.firstChild) {
      n.removeChild(n.firstChild);
}

The code gets the main outer div, and then it loops as long as there is a first child element. If there is one, it removes that child. By the time the loop terminates, there will not no child elements to remove.

The following code segment removes only the first child:

let main = document.getElementById("main");
let sub = document.getElementById("sub");
let something = main.removeChild(sub);

The following code segment removes only the first child as well:

let n = document.getElementById("sub");
if (n.parentNode)
      n.parentNode.removeChild(n);

The following code segment removes the first child and throws an error:

let main = document.getElementById("main");
let sub = document.getElementById("sub");
let something = main.removeChild(sub);
while(true)
      something = main.removeChild(sub);

Because the first child was removed, making additional calls to removeChild()throws the following error:

NotFoundError: Node was not found
