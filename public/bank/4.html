SubObjective:

Complete or debug code that uses assignment and arithmetic operators

From the following variable declarations, which one evaluates to false?

var x = 17 > 23 || 3 + 4 <= 7;
var y = 12^2 && +'5' > 2;
var z = 8 + 3*4 > 12 || 13;
var t = !9 || 4 + 6 - 2 < 3;

x
y
z
t

Explanation

Evaluating the expressions one at a time results in t being false and x, y, and z being true.

Starting with t:

9 is true (since a number except 0 is true). !9 is false. 4+6-2 is 8. 8 < 3 is false. Therefore, false || false is false.

For x:

17 > 23 is false. 3+4 is 7 and 7 <= 7 is a true statement. Therefore, false || true is true.

For y:

12^2 is 14. ^ is an XOR operator. When evaluating bits, an XOR b is true if a and b are different. 12 is 1100 in binary, and 2 is 0010 in binary. XORing the bits will lead to 14. The unary operator + will make the expression numeric. Therefore +'5' is equal to 5. Since 14 is true and 5> 2 is true, true && true is true.

For z:

The multiplication is evaluated before the addition. So 8 + 3*4 is 8 + 12 = 20. 20 > 12 is true.
13 is true. Therefore true || true is true.
