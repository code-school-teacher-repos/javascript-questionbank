Explanation

The following will show in the console:

The employees array contains objects that represent the name and salary of each employee:

var employees = [
{ name: 'James', salary: 10000 },
{ name: 'Bryan', salary: 7000 },
{ name: 'Tim', salary: 14000 },
{ name: 'Jessica', salary: 21000 },
{ name: 'Lisa', salary: 33000 },
{ name: 'tom', salary: 8000 }
];

The array is then sorted and printed in the console.

Note that the sort() function will work correctly with strings and numbers. However, when it comes to objects, you will need to provide a function that guides the sort in knowing what it means for objects to be equal. For example, two computer objects can be equal if the RAM is equal, and two shapes are equal if they have the same area.

In this case, the function passed to sort is trying to compare objects by name. If obj1 comes before obj2, it will return -1. If it comes after it, it will return 1, and if they are the same, it will return 0. a and b are the objects to compare. You start by reading the name of each object, convert it to lower case (or upper case), and then compare it with the others. With this function, sort() will be able to tell how to order the objects by name.

So when loping over the array and printing the objects, the result will be as follows (note that "T" comes before "t" in sorting order):

Name: Bryan, Salary: 7000.00
Name: James, Salary: 10000.00
Name: Jessica, Salary: 21000.00
Name: Lisa, Salary: 33000.00
Name: Tim, Salary: 14000.00
Name: tom, Salary: 8000.00
