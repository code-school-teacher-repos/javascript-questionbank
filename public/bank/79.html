Interact with the Document Object Model
SubObjective:

Identify and construct the Document Object Model (DOM) tree

When the page loads, you need to open a window with the following characteristics:

* The URL https://www.measureup.com loads in a new window.
* It is a 300x300 window.
* It is displayed in the top left corner from the main window.
* It does not have any of the following: menu bar, scroll bar, status bar, title bar, or tool bar.
* It is not resizable.

Which code should you use?

window.open("https://www.measureup.com", "_self", "height=300,width=300,left=100,top=50,menubar=0,resizable=0,titlebar=0,toolbar=no");
window.open("https://www.measureup.com", "_blank", "height=300,width=300,left=100,top=50,menubar=0,resizable=0,scrollbars=0,status=0,titlebar=0,toolbar=no");
window.open("https://www.measureup.com", "MeasureUp", "height=300,width=300,left=100,top=50,menubar=1,resizable=0,scrollbars=0,status=1,titlebar=0,toolbar=no");
window.open("https://www.measureup.com", "_blank", "height=300,width=300,left=100,top=50,menubar=0,resizable=1,scrollbars=yes,status=0,toolbar=1");

Explanation

You should use the following code:

window.open("https://www.measureup.com", "_blank",
"height=300,width=300,left=100,top=50,menubar=0,resizable=0,scrollbars=0,status=0,titlebar=0,toolbar=no");

The window.open() function has the following parameters:

* URL: The URL of the page to open. In this case, https://www.measureup.com.

* Name: This is the target of the window. It can be the name of the window, or one of the following: _blank, _parent, _self, or _top. Since _blank is the default, that means the URL is loaded into a new window. So, _blank is the one that will be used in this case. You should not use _self in this case because the URL will replace the current page.

* Specs: This is a comma separated list of items. For example, if you need a status bar in the new window, you can say:

status=yes or status=1

If you do not need a status, you can say:

status=no or status=0

The height and the width are in pixels, and they are provided as 300x300.

The location 100px from the left and 50px from the top satisfies the requirement.

Putting these properties together will result in:

"height=300,width=300,left=100,top=50,menubar=0,resizable=0,scrollbars=0,status=0,titlebar=0,toolbar=no".

Not all properties work perfectly well in all browsers, or in different versions of the same browser.
