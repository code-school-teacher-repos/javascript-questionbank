SubObjective:

Complete and debug code that uses built-in Math functions

You load the following page in the browser:

<!DOCTYPE html>
<html>
<head>
      <title>Intro to JavaScript</title>
           <meta charset="utf-8" />
</head>
<body>
      <script>
           var x = 10;
           var x1 = Math.ceil(Math.LN10);
           var x2 = Math.floor(Math.LN2);
           var x3 = Math.round(Math.log(x));
           var x4 = Math.round(Math.log2(x));

           console.log(`The value of Math.max() is ${Math.max(x, x1, x2, x3, x4, 0.7)}`);
           console.log(`The value of Math.min() is ${Math.min(x, x1, x2, x3, x4, 0.7)}`);
      </script>
</body>
</html>

What will be displayed in the console?

The value of Math.max() is 10 The value of Math.min() is 0.693
The value of Math.max() is 70 The value of Math.min() is 0
The value of Math.max() is 10 The value of Math.min() is 0
The value of Math.max() is 30 The value of Math.min() is 1

Explanation

When the page is loaded in the browser, the following is displayed in the console:

The value of Math.max() is 10
The value of Math.min() is 0

To start, multiple values are declared that use the Math properties and methods. Math.LN10 and Math.LN2 are properties with the following values:

Math.LN10: 2.302
Math.LN2: 0.693

Then you apply the ceil() and floor() on these values:

Math.ceil(Math.LN10): 3
Math.floor(Math.LN2): 0

Math.log() and Math.log2() find the natural logarithm and base 2 logarithm of a number respectively. The round() method will round these values to the nearest integer. Here is the outcome:

Math.round(Math.log(10)): 2
Math.round(Math.log2(10)): 3

Finally, the max() and min() functions are applied to find the largest and smallest values in the set:

Math.max(10 3 0 2 3 0.7): 10
Math.min(10 3 0 2 3 0.7): 0
