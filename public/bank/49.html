Program with Variables, Data Types, and Functions
SubObjective:

Complete and debug code that uses built-in Math functions

You are debugging the following code:

var a = Math.SQRT2;
var b = Math.E;
var c = Math.PI;
var d = Math.min(2.5, -3.2, 4, -5);
var e = Math.max(7, -2, 3.5, 1);
var values = [Math.round(a), Math.floor(b), Math.ceil(c), d, e];

getContent();

function getContent() {
console.log(values);
}

What content will be displayed for the values array in the console?

Array [ 1, 2, 4, -5, 7 ]
Array [ 2, 2.8, 3.2, 7, -3.2 ]
Array [ 1, 3, 3, -5, 7 ]
Array [ 2, 3, 4, 7, -5 ]

Explanation

The content of the array will be displayed as Array [ 1, 2, 4, -5, 7 ] in the console.

Going through the variables one by one will result in the following:

a is Math.SQRT2, which is 1.4142135623730951. Since the value after the decimal is 4 (below 5), Math.round(a) produces 1.

b is Math.E, which is a constant of value 2.718. The floor (the largest integer less than or equal to the number) of b is 2.

c is Math.PI, which is another constant representing 3.14159. The ceiling (the smallest integer greater than or equal to the number) of this number is 4.

In the set of values 2.5, -3.2, 4, and -5, the minimum is -5.

For the set 7, -2, 3.5, and 1, the maximum is 7.

The console will not display Array [ 1, 3, 3, -5, 7 ]. The first 3 is the ceiling of Math.E, not the floor. And the second 3 is the round of Math.PI, not the ceiling.

The console will not display Array [ 2, 3, 4, 7, -5 ]. First, rounding Math.SQRT2 is 1, not 2. In addition, the last two values (7 and -5) are reversed.

The console will not display Array [ 2, 2.8, 3.2, 7, -3.2 ]. The first value should be 1 instead of 2 because rounding Math.SQRT2 is 1. Also the floor of Math.E is 2 rather than 2.8. The ceiling of Math.PI is 4, not 3.2. The minimum and maximum are -5 and 7 correspondingly.
