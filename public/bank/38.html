You have the following HTML page:

<!DOCTYPE html>
<html>
<head>
      <title>Intro to JavaScript</title>
           <meta charset="utf-8" />
</head>
<body>
      <script>
           var letters = ["A", "B", "D", "F", "N", "R", "T", "C", "O", "P"];
           var x = letters.shift();
           var u = letters.splice(1, 3);
           letters.push("E");
           console.log(letters);
      </script>
</body>
</html>

What shows in the console when the page is loaded in the browser?

Array [ "B", "R", "T", "C", "O", "P", "E" ]
Array [ "A", "N", "R", "T", "C", "O", "P", "E" ]
Array [ "B", "N", "R", "T", "C", "O", "P", "E" ]
Array [ "A", "N", "R", "T", "C", "O", "E" ]

Explanation

The following will show in the console:

Array [ "B", "R", "T", "C", "O", "P", "E" ]

The letters array starts with the following content:

["A", "B", "D", "F", "N", "R", "T", "C", "O", "P"]

Then, the shift function removes the first element ("A"):

["B", "D", "F", "N", "R", "T", "C", "O", "P"]

The splice function removes n elements from a specific position. For example splice (2,4) removes 4 elements starting at index 2. In this scenario, splice(1,3) removes three elements starting at index 1:

["B", "R", "T", "C", "O", "P"]

Finally, the push function adds an element to the end of the array:

["B", "R", "T", "C", "O", "P", "E"]

The console.log statement writes the element in the console in a comma-separated list:

Array [ "B", "R", "T", "C", "O", "P", "E" ]
