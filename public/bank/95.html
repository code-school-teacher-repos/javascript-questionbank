Interact with the Document Object Model
SubObjective:

Complete and debug code that locates, modifies, and adds HTML elements and attributes

The HTML in a document starts as follows:

<ul id="myItems">
      <li><a href="Question10.html">Question 10</a></li>
      <li><a href="Question20.html">Question 20</a></li>
</ul>

Then, you run the following code:

<script>
      var inner = document.createElement("a");
      inner.setAttribute("href", "Question.html");
      var template = document.createElement("li");
      template.appendChild(inner);
      var list = document.getElementById("myItems");
      for (var i = 0; i < 5; i++) {
           var item = template.cloneNode(true);
           item.firstChild.setAttribute('href', `Question${i+1}.html`);
           item.firstChild.innerHTML = `Question ${i + 1}`;
           list.appendChild(item);
      }
</script>

You use the following style:

<style>
      ul {list-style-type:none;}
      ul li a {text-decoration: none;}
</style>

What will be displayed in the browser?

Question 1 Question 2 Question 3 Question 4 Question 5 Question 10 Question 20
Question 10 Question 1 Question 2 Question 3 Question 4 Question 5 Question 20
Question 10.html Question 1.html Question 2.html Question 3.html Question 4.html Question 5.html Question 20.html
Question 10 Question 20 Question 1 Question 2 Question 3 Question 4 Question 5

Explanation

After running the code in the script, the following will be displayed in the browser:

Question 10
Question 20
Question 1
Question 2
Question 3
Question 4
Question 5

You are starting with the first two list items: Question 10 and Question 20. The code starts by creating an <a> element (<a></a>). It sets its href to Question.html. An <li> element is created, and the <a> element is appended to it.

getElementById() retrieves the myItems unordered list. The loop iterates 5 times, and in each iteration the following happens:

* The <li> element called template is cloned with its content.
* The clone's href attribute and its innerHTML are set based on the Question text and the loop index i.
* The resulting <li> element is appended to the list.

When this process is done: Question 1, Question 2, Question 3, Question 4, and Question 5 are created. They are appended to the original Question 10 and Question 20 resulting in:

Question 10
Question 20
Question 1
Question 2
Question 3
Question 4
Question 5
