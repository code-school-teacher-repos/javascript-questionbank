You have the following code:

<script>
      var count = 0;

      function main() {
           try {
                --count;
                walkThrough();
           }
           catch (e) {
                count++;
           }
           finally {
                count++;
           }
      }

      function walkThrough() {
           try {
                var obj = {}
                if (obj)
                     count++;
                else {
                     let test = "This is a testing string";
                     if (test.indexOf('is') === 2)
                          count++;
                     else {
                          throw new Error("Error");
                     }
                }
           }
           catch (e) {
                count--;
           }
           finally {
                count++
           }
      }

      main();
      console.log(`Count: ${count}`);
</script>

What will be the value of count?

3
4
1
2

Explanation

The value of count will be 2.

count starts at 0. The call to main() decrements it to -1 and calls walkthrough(). In the try block, the object is initialized to empty, but this is still considered true, and count is incremented to 0. Since the finally block always executes, count is now 1.

There were no errors thrown, so the catch block in main() does not execute, but finally does, and count is incremented to 2.
