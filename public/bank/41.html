Program with Variables, Data Types, and Functions
SubObjective:

Declare, and use arrays

You have the following HTML page:

<!DOCTYPE html>
<html>
<head>
      <title>Intro to JavaScript</title>
           <meta charset="utf-8" />
</head>
<body>
      <script>
           var arr1 = ["Apple", "Orange", "Kiwi", "Lemon", "Banana"]
           var arr2 = ["Tomato", "Onion"];
           var arr3 = arr1.concat(arr2);
           arr3.shift();
           var arr4 = arr3.filter(function (item) {
                return item.length > 5;
           });
           var arr5 = arr4.map(function (e) {
                return e.toUpperCase();
           });
           console.log(arr5.reverse().join(","));
      </script>
</body>
</html>

What will be displayed in the console when the page loads?

TOMATO,ONION,LEMON,KIWI
ORANGE,BANANA,APPLE
BANANA,TOMATO,ORANGE
TOMATO,BANANA,ORANGE

Explanation

The following displays in the console:

TOMATO,BANANA,ORANGE

Starting with two arrays:

["Apple", "Orange", "Kiwi", "Lemon", "Banana"]
["Tomato", "Onion"]

The concat() function will produce:

[ "Apple", "Orange", "Kiwi", "Lemon", "Banana", "Tomato", "Onion" ]

Shifting will result in:

[ "Orange", "Kiwi", "Lemon", "Banana", "Tomato", "Onion" ]

The filter() function will return all the elements that meet the condition item.length > 5

[ "Orange", "Banana", "Tomato" ]

Then map() maps each element to upper case:

[ "ORANGE", "BANANA", "TOMATO" ]

Finally, the reverse() function reverses the elements and join() joins them with a comma (default):

TOMATO,BANANA,ORANGE
