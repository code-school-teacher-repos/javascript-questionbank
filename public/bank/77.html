Interact with the Document Object Model
SubObjective:

Identify and construct the Document Object Model (DOM) tree

You are writing code that needs to do the following:

* Ask the user for the number of records to display per page.
* Use that number as a part of a loop counter to display additional data.

Which statement should you use?

var count = window.alert("How many records do you need to display per page?");
var count = window.prompt("How many records do you need to display per page?");
var count = parseInt(window.prompt("How many records do you need to display per page?"));
var count = window.confirm("How many records do you need to display per page?");

Explanation

You need to use the following statement to ask the user for the number of records per page:

var count = parseInt(window.prompt("How many records do you need to display per page?"));

The prompt window displays a text box in which the user can enter a value. However, that value is always returned as a string. For example, if the user types 25, the return value is "25". Since this value will be used as a loop counter, it is important to convert it from string to numeric. The parseInt() function helps you accomplish that. Therefore, the following statement is not sufficient:

var count = window.prompt("How many records do you need to display per page?");

The following statement displays a window with an OK button and the user does not have a place to enter the value:

var count = window.alert("How many records do you need to display per page?");

Using window.confirm() will display the text with the OK and Cancel buttons, but the user is still not able to enter a number:

var count = window.confirm("How many records do you need to display per page?");
