What is the standard way of declaring a variable that holds the tax rate of 0.07?

var taxRate = 0.07;
let taxRate = 0.07;
taxRate = 0.07;
const taxRate = 0.07;

Explanation

The standard way of declaring a variable that holds the tax rate of 0.07 is:

const taxRate = 0.07;

Because the rate will rarely change, const is the best way to declare such variables so they will not be changed unintentionally.

Other ways of declaring variables involve using the let and var keywords. What is different about let is that it is block scoped, so it is only available within the block it is declared in. You should not use let and var in this scenario because of the nature of a tax rate being fairly constant.

Not using any keyword in front of the variable (as in taxRate = 0.07;) is not acceptable either because such declaration will make the variable global. Global variables are the source of many errors.
