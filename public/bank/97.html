Interact with the Document Object Model
SubObjective:

Complete and debug code that locates, modifies, and adds HTML elements and attributes

You have the following code segment:

<body>
      <div id="main1">You can prompt the user for input.</div>
      <script>
           let mainDiv = document.getElementById("main1");

           //Add code here </script>
</body>

You need to complete the code so it will display the following:

You can prompt the user for input.You can display output using an alert.

Which code segment should you use?

let element = document.createElement('span'); element.setAttribute("id", "main2"); element.textContent = "You can display output using an alert."; mainDiv.insertAdjacentElement("afterbegin", element);
let element = document.createElement('span'); element.setAttribute("id", "main2"); element.textContent = "You can display output using an alert."; mainDiv.insertAdjacentElement("beforeend", element);
let element = document.createElement('span'); element.setAttribute("id", "main2"); element.textContent = "You can display output using an alert."; mainDiv.insertAdjacentElement("afterend", element);
let element = document.createElement('p'); element.setAttribute("id", "main2"); element.textContent = "You can display output using an alert."; mainDiv.insertAdjacentElement("beforebegin", element);

Explanation

You should use the following code:

let element = document.createElement('span');
element.setAttribute("id", "main2");
element.textContent = "You can display output using an alert.";
mainDiv.insertAdjacentElement("beforeend", element);

The output places both text content instances on the same line:

You can prompt the user for input.You can display output using alert.

The function insertAdjacentElement() inserts the element in the second parameter based on the string provided in the first parameter:

beforebegin: Before the div tag begins.
afterbegin: After the tag begins, but before any content.
beforeend: Before the end of the tag.
afterend: After the tag ends.

If you insert the span after the tag ends, it will be placed on the next line. You will need to insert it before the tag ends.

Inserting the span after the tag begins will place the text backwards:

You can display output using an alert.You can prompt the user for input.

Using the <p> tag instead of the <span> will produce the following output:

You can display output using an alert.
You can prompt the user for input.
