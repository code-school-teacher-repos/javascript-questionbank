Implement and Analyze Decisions and Loops
SubObjective:

Complete and debug loops

You have the following code:

var count = 50;
var outcome = 0;
for (; ;count--) {
      if (count <= 15) break;
      if (count % 5 == 0) continue;
      outcome += count;
}
console.log(`Count: ${count}; Outcome: ${outcome}`);

What are the correct values for count and outcome?

Count: 15; Outcome: 1155
Count: 15; Outcome: 910
Count: 50; Outcome: 0
Count: 14; Outcome: 905

Explanation

The value of count will be 15. The value of outcome will be 910.

The for loop is made up of three sections: initialization, condition, and final expression. The condition is evaluated before each loop iteration. The final expression is evaluated at the end of each loop iteration:

for(initialization; condition; final expression)

These three expressions do not have to exist inside the parentheses of the for loop. Instead, they can be in the loop body (between { and }).

In this scenario, one expression is in the body of the loop and the second one is in the head:

for (; ;count--) {
      if (count <= 15) break;
      if (count % 5 == 0) continue;
      outcome += count;
}

The initialization section is before the loop (var count = 50). The condition is inside the loop (if (count <= 15) break;).

In each iteration, the condition is tested. The loop will exit when count is 15.

If count is divisible by 5 (count % 5 == 0), the continue statement will execute. That means "go to the next iteration of the loop by skipping the rest of the code". In other words, the statement outcome += count; is skipped whenever count is divisible by 5.

Going through each iteration and applying the conditions above, outcome will be 910.
