You have the following code:

<!DOCTYPE html>
<html>
<head>
      <title>Intro to JavaScript</title>
           <meta charset="utf-8" />
</head>
<body>
      <script>
           var outcome = ' ';

           var statement = "The quick brown fox jumps over the lazy dog";
           var items = statement.split(" ");
           for (var i = 0; i < items.length; i++) {
                takeAction(items[i]);
           }
           alert(outcome);

           function takeAction(item) {
                switch (item.toLowerCase()) {
                     default:
                          outcome += '_default';
                     case "over":
                          outcome += "_under";
                          break;
                     case "the":
                     case "dog":
                          outcome += "_doggy";
                     break;
                     case "jumps":
                          outcome += "_nice";
                     case "quick":
                     case "fox":
                          outcome += "_foxy";
                          break;
                }
           }

      </script>
</body>
</html>

What will be displayed in the alert() statement?

_doggy_foxy_default_under_foxy_nice_foxy_under_doggy_default_under_doggy
_doggy_foxy_default_under_foxy_nice_under_doggy_default_under_doggy
_doggy_foxy_under_foxy_nice_foxy_under_doggy_under_doggy
_default_default_default_default_default_default_default_default_default

Explanation

The alert() statement will display the following:

_doggy_foxy_default_under_foxy_nice_foxy_under_doggy_default_under_doggy

Using the split() call on the string results in an array of words. Each word in the array is then passed to the takeAction() function. The switch statement matches the word with its corresponding case. For that, an appropriate word is accumulated into outcome.

Note that the default case is placed at the beginning, and that is not an error, nor does it mean that the default case will run for every scenario. On the contrary, the default case runs as usual, which is when other cases do not match the item (in lower case). It is exactly like the else of an if-else if-else statement. For example, the default case will run for "brown" since that value matches none of the cases.

Also, it is important to note that the "jumps" case does not have a break statement. That means that the "quick" and "fox" cases will run as well.

As the loop passes the words to the switch statement, each word is handled, and for those that do not have matches, the default case will run. In the end, the following outcome will show up in the alert:

_doggy_foxy_default_under_foxy_nice_foxy_under_doggy_default_under_doggy
