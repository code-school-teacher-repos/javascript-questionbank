Which code snippet prints "Result: 31" in the console?

function getNumbers(x) { var result = ' '; for (var i = 0; i < x; i++) { result += Math.pow(2, i) + '  '; } return result; } console.log(`Result: ${getNumbers(5)}`);
function getRandom(a, b) { return Math.floor(Math.random() * (b - a + 1)) + a; } console.log(`Result: ${getRandom(5,20)}`);
console.log(`Result: ${Math.round(Math.ceil(Math.pow(Math.PI,2)) * Math.PI)}`);
console.log(`Result: ${Math.round(Math.ceil(Math.E) * Math.PI)}`);

Explanation

The call that prints "Result: 31" in the console is:

console.log('Result: ${Math.round(Math.ceil(Math.pow(Math.PI,2)) * Math.PI)}');

The value of Math.ceil(Math.pow(Math.PI,2)) is 10 since PI (3.14159) is raised to the power 2 (squared), and the ceiling(the smallest integer that is greater than or equal to the number) returns 10. Math.ceil(Math.pow(Math.PI,2)) * Math.PI is 31.416. Rounding this value to the nearest integer results in 31.. The console.log() prints "Result: 31".

The following call results in 9. The order of operations states that E (2.718) has the ceiling calculated first, then multiplied by PI (3.14159), and the outcome is rounded.

Math.round(Math.ceil(Math.E) * Math.PI)

You should not use the function getRandom()because it generates a random number that falls between the parameters provided, inclusive:

function getRandom(a, b) {
return Math.floor(Math.random() * (b - a + 1)) + a;
}

First, Math.random() generates a random number between 0 and 1. Multiplying this number by (b - a + 1)) + a results in a value between a and b. Please note that since this a random number, running this code results in a different number every time. That number, however, falls between 5 and 20 inclusive. Therefore, it will not produce 31.

You should not use the function getNumbers(5) because it loops five times and accumulates the values of "2 to the power of the loop index" in result. The outcome is 1 2 4 8 16.

function getNumbers(x) {
var result = '';
for (var i = 0; i < x; i++) {
result += Math.pow(2, i) + ' ';
}
return result;
}
