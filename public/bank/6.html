SubObjective:

Complete or debug code that uses assignment and arithmetic operators

What will be printed in the console after the following statements are evaluated?

var a = 17;
var b = 21;
var c = b << 1, d = a >> 1;
if (c && d)
      console.log(c | d);
else
      console.log(c & d);

21
42
4
8

Explanation

42 will be printed in the console. << and >> are the left shift and right shift operators respectively. << shifts the operand a number of bits to the left and >> shifts the operand a specified number of bits to the right. For example, b << 1 shifts b (which is 21) 1 bit to the left. The excess bits shifted from the left are discarded and the 0 bits are shifted in from the right. 21<<1 is 00010101 << 1. This evaluates to 42.

In the same manner, 17>>1 is 00010001 >> 1, which is 8.

Since c and d are both non-zero values, the condition (c && d) evaluates to true, and the first console.log statement is executed.

c | d = 42 | 8 = 00101010 | 1000 = 42, and that is what gets printed in the console.
