Explanation

The following will be displayed on the screen:

The standard chunk of Lorem Ipsum used since the 1500s is reproduced below for those interested.

The document.write() function writes text directly to the HTML document. In this scenario, the function showText() is called when the page loads, which causes all existing HTML to be deleted. The other divs (with their content) will be replaced with the one line from document.write() in showText().

Other options are incorrect because they show the text that the document.write() has replaced.
