Implement and Analyze Decisions and Loops
SubObjective:

Evaluate expressions that use logical and comparison operators

You have the following declarations:

var x = new String('welcome to JavaScript');
var y = new String('welcome to JavaScript');

Which expression would evaluate to true?

x === y
x == 'welcome to JavaScript'
x === 'welcome to JavaScript'
x == y

Explanation

The expression that is true is:

x == 'welcome to JavaScript'

In this scenario, you are using the == operator, which means that type conversion will take place. The object x is converted to 'welcome to JavaScript' and the string comparison returns true.

The expression x === 'welcome to JavaScript' is false because the === operator compares content and type. Since x is an object and 'welcome to JavaScript' is a string literal, the result is false.

x == y and x === y are both false because x and y are objects referencing different locations in memory and for that reason they will not be equal.
