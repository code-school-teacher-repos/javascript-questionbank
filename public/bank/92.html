When the page is loaded in the browser, you need the following to be displayed:

Lorem Ipsum has been the industry's standard dummy
Lorem Ipsum is that it has a more-or-less normal distribution
over the years, sometimes by accident
There are many variations of passages

Which statement should you use?

document.getElementById('variation').style.display = "block";
document.getElementById('variation').style.display = "none";
document.getElementById('variation').style.display = "inline-block";
document.getElementById('variation').style.display = "inline";

Explanation

The code that helps you accomplish the task is:

document.getElementById('variation').style.display = "block";

The 'variation' element is a <span>, and spans flow inline with the other elements. In other words, if the display property is set to 'inline', the default, the content will flow as follows:

Lorem Ipsum has been the industry's standard dummy
Lorem Ipsum is that it has a more-or-less normal distribution
over the years, sometimes by accident There are many variations of passages Contrary to popular belief, Lorem Ipsum is not simply random text

For the span to display like a div and have its own block, you need to set the display property to "block".

Setting the display to "inline-block" means that the element itself is rendered as a block, but the other adjacent inline elements will be displayed on the same line.

Setting the display property to "none" will not display the element, and the rest of the content would be:

Lorem Ipsum has been the industry's standard dummy
Lorem Ipsum is that it has a more-or-less normal distribution
over the years, sometimes by accident Contrary to popular belief, Lorem Ipsum is not simply random text
