You have the following code:

<script>
      var letters = ['p', 'k', 'a', 'o', 'u', 'e', ' ', 'i', 'j', 'n', 'b'];
      var result = [ ];

      while (true) {
           var t = letters.shift();
           if (t === 'e')
                break;
           evaluate(t);
      }
      alert(result.length);

      function evaluate(x) {
           switch (x) {
                case 'a':
                case 'e':
                case 'i':
                case 'o':
                case 'u':
                     result.push(x);
                default:
                     if (result.length > 0)
                result.pop();
           }
      }
</script>

What value will be displayed in the alert when the code executes?

0
1
4
2

Explanation

The value that will be displayed in the alert is 0.

The script starts with two arrays: one for the letters and the other for the result. The while loop is an infinite one (true in the condition is always true). Letters are shifted (removed from the start) of the array one by one and passed to the evaluate function. The condition that the letter is 'e' will cause the infinite loop to break.

In the evaluate() function, the switch statement checks each letter: vowels are pushed into the result array and other letters are popped (using the default case). The issue, however, is that the vowels block does not have a break statement. That means that the default case is executed as well every time. The letter just added to the array is being removed (popped out).

With that outcome, the result array will end up with no letters, and the length is 0.
