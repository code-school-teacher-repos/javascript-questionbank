Which variable declaration is not valid?

let _grades = [];
var Course_98_382_Question_Count = 75;
const $studentCount = 34;
var 98_382_grades = [];

Explanation

The following variable declaration is not valid:

var 98_382_grades = [];

Variables hold values in a program. You can declare variables to hold the total balance, student count, the number of records to process and so on. Variables are also called identifiers. A variable declaration must start with a letter, a dollar sign, or an underscore. Subsequent characters can be digits. To declare a variable, the keywords var, const, and let are used.

In this scenario, 98_382_grades is not a valid declaration because it starts with a number instead of a letter.

Course_98_382_Question_Count is a valid declaration because it starts with a letter. Digits after the first letter are valid. Along the same lines, $studentCount and _grades are also valid since they start with $ and _ respectively.
