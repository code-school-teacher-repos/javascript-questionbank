Program with Variables, Data Types, and Functions
SubObjective:

Complete and debug a function that accepts parameters and returns a value

You load the following page in the browser:

      <script>
           let beeCount = 400;
           let course = {
                name: 'intro to programming',
                id: 1012,
                description: 'this course introduces students to programming',
                students: [
                     student1 = {
                          name: 'James',
                          credits: 5
                     },
                     student2 = {
                          name: 'Bryan',
                          credits: 3
                     }
                ]
           };

           var func1 = function (a) {
                a = 456;
           }

           var func2 = function (item) {
                if (!item) return;
                item.name = 'JavaScript programming';
                item.id = 1015;
           }

           function callThem() {
                console.log(beeCount);
                func1(beeCount);
                console.log(beeCount);

                console.log(`Name: ${course.name}, id: ${course.id}`);
                func2(course);
                console.log(`Name: ${course.name}, id: ${course.id}`);
           }

           callThem();
      </script>

What will be displayed in the console?

400 400 Name: intro to programming, id: 1012 Name: JavaScript programming, id: 1015
400 456 Name: intro to programming, id: 1012 Name: JavaScript programming, id: 1015
456 456 Name: intro to programming, id: 1012 Name: JavaScript programming, id: 1015
400 400 Name: intro to programming, id: 1012 Name: intro to programming, id: 1012

Explanation

The following will be displayed in the console:

400
400
Name: intro to programming, id: 1012
Name: JavaScript programming, id: 1015

Notice that beeCount is a number while course is an object. Basic types like a number are passed to a function by value. That means that only a copy of the variable is passed, not the variable itself. Changes are made to the copy, and the original variable is not impacted. In the following segment:

console.log(beeCount);
func1(beeCount);
console.log(beeCount);

The original value of beeCount (400) is printed to the console. Then a copy is passed to the function. The copy is changed inside the function to 456. But, by the time the function terminates, that copy is out of scope and beeCount retains its original value of 400.

On the other hand, course is passed to func2(). Objects are passed by reference. That means that a memory reference to the object course is passed in, and any changes made to the object are permanent.

console.log(`Name: ${course.name}, id: ${course.id}`);
func2(course);
console.log(`Name: ${course.name}, id: ${course.id}`);

The original name and id are printed. Then, the same course object is passed by reference to the function func2(). The function changes the actual values permanently, and now the new name and id are printed.
