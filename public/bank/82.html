Identify and handle HTML events

You are writing code so that when the user selects a language from a drop-down menu, the corresponding value will be displayed in the alert.

You need to complete the code.

Which code snippet should you use? To answer, select the appropriate option from the drop-down menu.

Explanation

The missing code from the fragment is as follows:

onchange="handleEvent(event);"

This code handles the onchange event in the HTML. When the option in the drop-down changes, the event will fire and call the handleEvent(event) by passing the event information. In the event body, the event's target value contains the value of the HTML <option> being selected. That value is displayed in the alert.

In the following option, the event function will not be called and no alert will be displayed on the screen because the event information is not being passed as a parameter:

onchange="handleEvent();"

The onkeypress is not correct for this scenario. The event will not fire when the user presses keys related to the text in the drop-down. Key press events are mostly related to input boxes (<input type="text" onkeypress="func()">).

Using onselect will not cause the event to fire and display the value from the corresponding option:

onselect="handleEvent(event);"

This event works mostly with input elements and text areas.
