Program with Variables, Data Types, and Functions
SubObjective:

Declare and use variables of primitive data types

You have the following HTML page:

<!DOCTYPE html>
<html>
<head>
      <title>Intro to JavaScript</title>
           <meta charset="utf-8" />
</head>
<body>
      <script>
           let str = "This is a test string. It illustrates some examples of using strings with JavaScript";

           function stringOps() {
                var result = str.substring(0, str.indexOf('of')).substring(str.indexOf('.') + 1).replace('some', 'many');
                console.log(result.trim().replace(/ /g, '|').toUpperCase());
           }

           stringOps();
      </script>
</body>
</html>

What will be displayed in the console when the page is loaded in the browser?

IT|ILLUSTRATES|MANY|EXAMPLES
IT|ILLUSTRATES
USING|STRINGS|WITH|JAVASCRIPT
THIS IS A TEST STRING

Explanation

The following will show in the console:

IT|ILLUSTRATES|MANY|EXAMPLES

The script starts with a declaration of the variable str. Note that there are three different ways of declaring a variable: using let, const, or var. In this case, the variable is declared using the let keyword.

Then the function stringOps()uses string operations to manipulate the string. The first substring reads from the beginning of the string up to the index of "of". This results in "This is a test string. It illustrates some examples of". Then, the second substring slices from one character after the index of the period to the end of the previous string, resulting in " It illustrates some examples of". Then the word "some" is replaced with "many". Then the string is trimmed from both sides (spaces are removed). The replace call replaces the space with a pipe character (|). The letter 'g' indicates that the replacement is global (across the whole string, not just the first occurrence). Finally, the string is turned to upper case.
