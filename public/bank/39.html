Program with Variables, Data Types, and Functions
SubObjective:

Declare, and use arrays

You have the following HTML page:

<!DOCTYPE html>
<html>
<head>
      <title>Intro to JavaScript</title>
           <meta charset="utf-8" />
</head>
<body>
      <script>
           var elements = [2, 4, 6, 8, 10, 12];
           elements.fill(3, 4);
           elements.shift();
           elements.unshift(5);
           elements = elements.sort();
           console.log(elements);
      </script>
</body>
</html>

What shows in the console when the page loads?

Array [ 4, 4, 4, 5, 6, 8 ]
Array [ 3, 4, 4, 5, 6, 10 ]
Array [ 3, 3, 4, 5, 6, 8 ]
Array [ 5, 5, 6, 8, 10, 12 ]

Explanation

The following will show in the console:

Array [ 3, 3, 4, 5, 6, 8 ]

The arrays starts as follows:

[2, 4, 6, 8, 10, 12]

The fill() function takes different parameters. In this case, it is fill(value, start). The value to fill is 3 and that starts at location 4. So, 3 will be the content from location 4 to the end of the array:

[2, 4, 6, 8, 3, 3]

The shift() function removes the first element:

[4, 6, 8, 3, 3]

The unshift() function adds "5" to the beginning of the array:

[5, 4, 6, 8, 3, 3]

The sort() function sorts the elements in the array:

[3, 3, 4, 5, 6, 8]

Finally, the console.log() function writes the content to the console:

Array [ 3, 3, 4, 5, 6, 8 ]
