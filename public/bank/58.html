Program with Variables, Data Types, and Functions
SubObjective:

Complete and debug a function that accepts parameters and returns a value

You are asked to investigate the following code:

<script>

      function takeAction(a, b, c) {
           console.log(arguments.length + ': ' + Array.from(arguments));
      }

      var obj = {
           first: 'John',
           last: 'Doe',
           level: 'expert'
      };

      var languages = ['JavaScript', 'C#', 'Java', 'C++', 'Python'];
      let x = 34;
      let y = undefined;
      let z = null;
      let a = true;
      let b = 'wow';
      let c = NaN;

      takeAction(a, b, c);
      takeAction(obj, x, y, c);
      takeAction(languages, x, a, z);

</script>

What does the last line print?

4: JavaScript,C#,Java,C++,Python,34,true,
4: JavaScript,C#,Java,C++,Python,34,true,null
undefined: undefined
8: JavaScript,C#,Java,C++,Python,34,true,null

Explanation

The last line in the code will print the following:

4: JavaScript,C#,Java,C++,Python,34,true,

After the following members have been declared, the function takeAction() is called three times with different set of parameters:

var obj = {
      first: 'John',
      last: 'Doe',
      level: 'expert'
};

var languages = ['JavaScript', 'C#', 'Java', 'C++', 'Python'];
let x = 34;
let y = undefined;
let z = null;
let a = true;
let b = 'wow';
let c = NaN;

The body of the function uses the arguments object, which is a local variable available to every function. This object has an entry for each argument that is passed to the function. You can tell the number of arguments by using arguments.length. However, arguments is an object, not an array. To list the arguments passed to the function, you use Array.from(arguments).

In the example above, the languages array, x, a, and z are passed to the function. Although the array elements are listed in a comma-separated list, these still considered 4 arguments. x is 34, a is true and z is null. null will not be printed, however. That is why the list finishes with a comma:

4: JavaScript,C#,Java,C++,Python,34,true,
