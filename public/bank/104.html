* Include a textbox where the user can enter a search query. The text can be easily deleted by clicking on the 'x' on the right corner of the textbox.

Explanation

You should use the following form:

<form method="post">
      <div>
           <label for="query">Search query:</label>
           <input type="search" id="query" name="query">
      </div>
      <div>
           <label for="textToSearch">Text to search:</label>
           <textarea id="textToSearch" name="textToSearch" cols="40" rows="15"></textarea>
      </div>
      <div>
           <input type="button" onclick="searchIt()" value="Submit">
      </div>
</form>

<script>
      function searchIt() {
           var query = document.forms[0].query.value;
           var textToSearch = document.forms[0].textToSearch.value;
           if (textToSearch.includes(query))
                alert(`${query} was found in the text`);
           else
                alert(`Could not find ${query} in the text`);
      }
</script>

Based on the following requirements:

Include a textbox where the user can enter a search query. The text can be easily deleted by clicking on the 'x' on the right corner of the textbox: To delete the text easily using the 'x' on the right corner, the type "search" should be used with the input box:

<input type="search" id="query" name="query">

Include a textbox with multiple columns and rows where the user can enter the text to be searched: To give the user the flexibility to type more text and support scrolling, a text area is needed:

<textarea id="textToSearch" name="textToSearch" cols="40" rows="15"></textarea>

Include a button that can be used to search for the query in the text to be searched box. A message will be displayed accordingly: The function to be called needs to retrieve the form values first. document.forms is a collection of forms that can exist on the page. Since there is only one form, you will use document.forms[0]. The value property is needed to get the value entered by the user:

var query = document.forms[0].query.value;
var textToSearch = document.forms[0].textToSearch.value;

The function includes() is then used for the search:

if (textToSearch.includes(query))
      alert(`${query} was found in the text`);
else
      alert(`Could not find ${query} in the text`);
}
