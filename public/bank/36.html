Program with Variables, Data Types, and Functions
SubObjective:

Declare, and use arrays

You have the following HTML page:

<!DOCTYPE html>
<html>
<head>
      <title>Intro to JavaScript</title>
           <meta charset="utf-8" />
</head>
<body>
      <script>
           var elements = ["green", "red", "yellow", "purple", "orange", "white"];
           var e1 = elements[elements.length - 1];
           var e2 = elements.unshift(e1);
           var e3 = elements.push("black");

           elements.forEach(function (item, i, arr) {
                if (item.length === 5)
                     console.log(item, i);
           });
      </script>
</body>
</html>

What will be displayed in the console when the page is loaded in the browser?

green 0 white 2 white 7 black 4
white 0 green 1 white 6 black 7
white 3 green 4 white 6 black 5 black 2
black 2 green 3 green 1 black 7

Explanation

The output will be as follows:

white 0
green 1
white 6
black 7

The elements array is declared with 6 color strings. elements.length represents the length of the array, which is 6. e1 is the element at location 5, which is the last element in the array since array indexing starts at 0.

unshift(e1) means adding e1 to the beginning of the array. Then, "black" is pushed to the end of the array. At this point, the array contains the following:

["white", "green", "red", "yellow", "purple", "orange", "white", "black"]

The last set of code loops over the elements of the array using forEach, and checks if the element's length is 5. If it is the case, the element and its index are printed in the console.
