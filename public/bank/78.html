You are creating a web application. Ten seconds after the application loads, you need to send information to the server about the health of the application.

Which code set should you use?

<body onload="takeAction()">       <script>            var timeoutId;            function takeAction() {                 timeoutId = window.setInterval(sendInfo, 10);            }            function sendInfo() {                 alert("Sending the application's health information to the server");            }       </script> </body>
<body onload="takeAction()">      <script>            var timeoutId;            function takeAction() {                 timeoutId = window.setInterval(sendInfo, 10000);            }            function sendInfo() {                 alert("Sending the application's health information to the server");            }      </script> </body>
<body onload="takeAction()">      <script>           var timeoutId;           function takeAction() {                timeoutId = window.setTimeout(sendInfo, 10000);            }           function sendInfo() {                alert("Sending the application's health information to the server");            }      </script> </body>
<body onload="takeAction()">      <script>            var timeoutId;            function takeAction() {                 timeoutId = window.setTimeout(sendInfo, 10);            }            function sendInfo() {                 alert("Sending the application's health information to the server");            }      </script> </body>

Explanation

You should use the following code set:

<body onload="takeAction()">
      <script>
           var timeoutId;
           function takeAction() {
                timeoutId = window.setTimeout(sendInfo, 10000);
           }

           function sendInfo() {
                alert("Sending the application's health information to the server");
           }
      </script>
</body>

The function that you should call is setTimeout(). With this function, you can set a delay on executing another function. The function to call is the first parameter, and the delay is the second parameter:

window.setTimeout(sendInfo, 10000);

sendInfo() is represented as an alert in this case, but this is the function that will make a call to the server and post all the health information. The delay parameter is in milliseconds. Since the requirement is for a 10 second delay, that will be represented as 10*1000 = 10000 milliseconds.

The takeAction() function is being called as the page loads:

<body onload="takeAction()">

You should not use setInterval() in this scenario because the call will take place every 10 seconds, while the requirement states that the information needs to be sent 10 seconds after the page loads (and only once):

window.setInterval(sendInfo, 10000);

You should not use calls to setTimeout()where the delay is specified as 10 because that represents 10 milliseconds, not 10 seconds.
