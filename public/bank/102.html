Interact with HTML Forms
SubObjective:

Complete and debug code that retrieves input from forms and sets form field values

You have an HTML page that contains the following markup:

<form id="student" method="post">
      <fieldset>
           <legend>Basic Info</legend>
           <div>
                <label for="first">First Name:</label>
                <input type="text" id="first" name="firstName">
           </div>
           <div>
                <label for="last">Last Name:</label>
                <input type="text" id="last" name="lastName">
           </div>
           <div>
                <label for="mainEmail">Email:</label>
                <input type="email" id="mainEmail" name="studentEmail">
           </div>
      </fieldset>
      <fieldset>
           <legend>Contact</legend>
           <div>
                <label for="parent">Parent Name:</label>
                <input type="text" id="parent" name="parentName">
           </div>
           <div>
                <label for="parentEmail">Parent Email:</label>
                <input type="email" id="parentmail" name="parentEmail">
           </div>
      </fieldset>
      <fieldset>
           <legend>Course</legend>
           <div>
                <label for="Id">Id:</label>
                <input type="text" id="Id" name="courseId">
           </div>
           <div>
                <label for="info">Detail:</label>
                <textarea id="info" name="courseInfo"></textarea>
           </div>
      </fieldset>
</form>

<form id="miscInfo">
      <fieldset>
           <legend>Misc Info</legend>
           <div>
                <label for="feedback">Feedback:</label>
                <textarea id="feedback" name="feedback" placeholder="Feedback on registration experience"></textarea>
           </div>
      </fieldset>
<form>
      <hr />
      <input type="button" onclick="getValues()" value="Submit">

When the submit button is clicked, you need the form to inform the user that the information on the course will be emailed to the email address provided.

Which code should you use?

function getValues() {      var myForm = forms[1];      var result = `Your registration for ${myForm.courseId.value} will be emailed to ${myForm.mainEmail.value}`;      alert(result); }
function getValues() {      var myForm = document.forms[0];      var result = `Your registration for ${myForm.courseId} will be emailed to ${myForm.studentEmail}`;      alert(result); }
function getValues() {      var result = `Your registration for ${getElementById("courseId").value} will be emailed to ${getElementByTagName("email").value}`;      alert(result); }
function getValues() {      var myForm = document.forms[0];      var result = `Your registration for ${myForm.courseId.value} will be emailed to ${myForm.studentEmail.value}`;      alert(result); }

Explanation

You should use the following code:

function getValues() {
      var myForm = document.forms[0];
  var result = `Your registration for ${myForm.courseId.value} will be emailed to ${myForm.studentEmail.value}`;
      alert(result);
}

document.forms is a collection of the forms on the page. Currently, there are two. Using document.forms[0] will return the first form. To retrieve the values of the input elements, you use the value as follows:

var myForm = document.forms[0];
myForm.courseId.value

The courseId is the name of the input box that contains the id of the course:

<input type="text" id="Id" name="courseId">

The same approach is used for retrieving the email: myForm.studentEmail.value

You should not use forms[1] because that will return the second form on the page (the feedback form).

Using the name of the field without the 'value' attribute will not return the entered data. So, the following is incorrect:

var result = `Your registration for ${myForm.courseId} will be emailed to ${myForm.studentEmail}`;

The following is incorrect as well because the getElementById()uses the wrong id (the course id is 'Id', not 'courseId'), and getElementByTagName() will fail because there are no tags named 'email'. There are input tags, but the type is email.

var result = `Your registration for ${getElementById("courseId").value} will be emailed to ${getElementByTagName("email").value}`;
alert(result);
