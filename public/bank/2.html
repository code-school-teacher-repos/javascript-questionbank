Interact with HTML Forms
SubObjective:

Complete and debug code that retrieves input from forms and sets form field values

You are creating a form that requests students' information. The code is as follows:

<div>
      <label for="name">Student Name:</label>
      <input type="text" id="studentName" name="studentName">
</div>
<div>
      <label for="Id">Course Id:</label>
      <input type="text" id="Id" name="courseId" placeholder="CSC-100">
</div>

You need to display the form as shown in the exhibit.

Which style should you use?

label {      display: block;      width: 150px;      text-align: right; }
.label {      display: inline-block;      width: 40px;      text-align: right; }
label {      display: inline-block;      width: 150px;      text-align: right; }
label {      display: block;      width: 150px;      text-align: left; }

Explanation

You should use the following CSS to style the content to the way it is shown in the exhibit:

label {
      display: inline-block;
      width: 150px;
      text-align: right;
}

The CSS above indicates that the styling (display, width, and text-align) targets the labels. The three key elements are the display, width, and text-align.

display: It is important for the display to be inline-block for the label and the input to be side-by-side. Setting the display to block will cause the label and the input to be on separate lines.

width: Enough width is needed for the Student Name to stay in the same line. 150px is wide enough. Using 40px as one of the options suggests is not enough and that will cause the label to be split on two lines.

text-align: The text needs to be aligned to the right to match the exhibit.

You should not use .label instead of label (notice the period). The period in CSS indicates a class name to be used in the HTML. The code above does not show any CSS class being used.
