SubObjective:

You have the following HTML page:

<!DOCTYPE html>
<html>
<head>
      <title>Intro to JavaScript</title>
           <meta charset="utf-8" />
</head>
<body>
      <div id="result"></div>
      <noscript>This is interesting!</noscript>
      <div>I would like to know the difference between let and var!</div>
      <script>

           var content = " @@@@This&is&a&test@@@@ ";
           document.getElementById("result").innerHTML = manipulateData(content);

           function manipulateData(text) {
                return text.trim().replace(/@/gi, "_").replace(/&/gi, "|").toUpperCase();
           }

      </script>
</body>
</html>

You decide to disable JavaScript in the browser.

What will be displayed on the screen when you load this page in the browser?

This is interesting!
@@@@This&is&a&test@@@@ This is interesting! I would like to know the difference between let and var!
____THIS|IS|A|TEST____ I would like to know the difference between let and var!
This is interesting! I would like to know the difference between let and var!

Explanation

The following will be displayed in the browser:

This is interesting!
I would like to know the difference between let and var!

You use the <noscript> tag to display content as an alternative when the JavaScript script blocks will not execute because JavaScript is turned off in the browser.

Since the div with id "result" will only display content from the execution of the script, this div will not render any content.

The next statement is the <noscript> block. The content of this block will render on the screen. You use this to inform the user that JavaScript is disabled and show different alternatives such as a link that displays only text when the user clicks it.

The div after the <noscript> block will render its content as well because it does not have any JavaScript code.
