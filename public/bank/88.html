Interact with the Document Object Model
SubObjective:

Complete and debug code that outputs to an HTML document

The body of the page contains the following code:

<body>
      <p>If you build</p>
      <p>the application</p>
      <p>they will come</p>
      <script>
           document.open();
           document.write("<p>and say <b>Hello World!<b></p>");
           document.close();
      </script>
</body>

What will be displayed on the screen when the page is loaded in the browser?

If you build the application they will come
and say Hello World!
If you build the application they will come and say Hello World!
and say Hello World! If you build the application they will come

Explanation

When you load the page in the browser, the following will be displayed on the screen:

If you build

the application

they will come

and say Hello World!

The original text in the paragraphs will remain. There is no onload function here to replace the page content. After the three paragraphs are written, the script opens the output stream, adds the text, and closes the output stream.

The options that list part of the text to be displayed are incorrect because none of the text is overwritten. The first three paragraphs are written, and then the text from document.write().
