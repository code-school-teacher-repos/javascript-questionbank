Program with Variables, Data Types, and Functions
SubObjective:

Complete and debug code that uses objects

A date is declared as follows:

var d2 = new Date('March 11, 2005 02:30:00');

How should you complete the following statements? To answer, select the appropriate options from the drop-down menus.

Explanation

The getDate() function displays 11. The getDate() function returns the day of the month (1 - 31). With the current declaration of the Date object, that will be 11.

The getUTCDate() function also displays 11. The getUTCDate() function behaves the same way in returning the day of the month according to universal time. This will be 11 as well.

The getMonth() function returns 2. The getMonth() function returns the month of the date specified according to local time. However, January starts at 0, which means the return value is 2, not 3 for March.

To display '2005-03-11T07:30:00.000Z', you need to call toISOString(). In order to get the long formatting of '2005-03-11T07:30:00.000Z', the toISOString() function needs to be called. The toLocaleDateString() and toLocaleTimeString() functions return '3/11/2005' and '2:30:00 AM' respectively.
