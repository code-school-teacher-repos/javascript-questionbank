Implement and Analyze Decisions and Loops
SubObjective:

Evaluate expressions that use logical and comparison operators

For each of the following expressions, select truthy if the statement evaluates to 'truthy'. Select falsey if the statement evaluates to 'falsey'.

Explanation

The expressions that evaluate to 'truthy' are:

{x: -0} ? 'truthy' : 'falsey'
1 == '1' ? 'truthy' : 'falsey'

Objects evaluate to true. So, {x: -0} is truthy. Even if x has a value of 0.
1 == '1' returns truthy as well because '1' is converted to a numeric value and 1 == 1.

Note the use of the ?: operator. It works as follows:

(expression1) ? (expression2 : (expression3)
expression1 is evaluated. If the result is true, the return value is the result of evaluating expression2. Otherwise, the result is the evaluation of expression3. This is a shortcut for the if-else statement.

1 === '1' will be falsey because the === operator tests for content and type.

undefined and null are always falsey. So are empty strings ('' and "") and 0 (even if that is +0 or -0).
