Interact with the Document Object Model
SubObjective:

Identify and construct the Document Object Model (DOM) tree

An HTML document contains a title and one script in the body.

You need to add code to write the title of the page and the content of the script and change the background color of the page to yellow.

Which code should you use?

window.document.write(`title: ${window.document.head.title}<br />`); window.document.write(`Scripts: ${window.document.scripts.text}<br />`); window.document.body.style.backgroundColor = 'yellow';
window.document.write(`title: ${window.document.title}<br />`); window.document.write(`Scripts: ${window.document.scripts[0].text}<br />`); window.document.body.style.backgroundColor = 'yellow';
window.document.write(`title: ${window.document.title}<br />`); window.document.write(`Scripts: ${window.document.scripts.length}<br />`); window.document.body.style.color = 'yellow';
window.document.write(`title: ${window.document.title}<br />`); window.document.write(`Scripts: ${window.document.scripts.toString()}<br />`); window.document.body.style.backgroundColor = 'yellow';

Explanation

The set of code that meets the requirements is:

window.document.write(`title: ${window.document.title}<br />`);
window.document.write(`Scripts: ${window.document.scripts[0].text}<br />`);
window.document.body.style.backgroundColor = 'yellow';

The document object has a write (or writeln) function that writes the content on the screen. It also has the title property that contains the title of the page. The scripts property of the document object contains a collection of all the scripts on the page. Since there is only one script per the requirements, that script can be gotten using scripts[0].

The background color is part of the document's body style. To set that value, you should use document.body.style.backgroundColor.

The following code does not print the title of the page and the script will be undefined because it is not subscripted before retrieving the text:

window.document.write(`title: ${window.document.head.title}<br />`);
window.document.write(`Scripts: ${window.document.scripts.text}<br />`);
window.document.body.style.backgroundColor = 'yellow';

The following code will print the title and change the background color, but will not print the script content:

window.document.write(`title: ${window.document.title}<br />`);
window.document.write(`Scripts: ${window.document.scripts.toString()}<br />`);
window.document.body.style.backgroundColor = 'yellow';

Instead, the following will be displayed:

Scripts: [object HTMLCollection]

This following code shows the title of the page as well as the number of scripts (1) in yellow text, but does not set the background color or show the script content:

window.document.write(`title: ${window.document.title}<br />`);
window.document.write(`Scripts: ${window.document.scripts.length}<br />`);
window.document.body.style.color = 'yellow';
