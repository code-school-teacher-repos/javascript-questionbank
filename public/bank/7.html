SubObjective:

Complete or debug code that uses assignment and arithmetic operators

The body of an HTML page contains the following code:

<div id="result" class="format-output"></div>
      <script>
           var a = 13, b = 43, c = 39, d = undefined;
           if (b && c) {
                d = (b + c) % (++a - 4)
           }
           else {
                var f = c-- - --b;
                if (f%2 == 0)
                     document.getElementById("result").innerHTML = 'success!' + f;
           }
           document.getElementById("result").innerHTML = d;
      </script>

What will be displayed in the browser after the page is loaded?

undefined
success!-1
1
2

Explanation

2 will be displayed in the browser after the page is loaded . b and c are non-zero values, and the condition b && c in the if statement will evaluate to true. Therefore, the if block will execute, and the else block will not.

To evaluate d, you need to know b+c and ++a. b + c = 43+39 = 82 and ++a = ++13 = 14.

d = (b+c) % (++a - 4) = 82 % (14 - 4) = 82 % 10 = 2.

Therefore, document.getElementById("result").innerHTML = d; will display 2 on the page.

Since d was evaluated, the default value of "undefined" will not be printed.

The variable f evaluates to -1. But since the else block is not executed, "success!-1" will not display.
