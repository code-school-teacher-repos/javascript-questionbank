SubObjective:

A student object is declared as follows:

var students = {
      "James": 300,
      "Bryan": 350,
      "Tim": 430,
      "Mike": 500,
      "Haley": 250,
      "Leah": 600,
      "Liz": 550,
      "Tom": 150,
};

You need to create a loop that can be used to accumulate the students' points.

Which type of loop should you create?

for...in
for
do...while
while

Explanation

When working with objects, the most suitable loop is the for...in loop. With this loop, you can easily iterate over the enumerable properties of the object.

Here is an example that illustrates the use of the for...in loop to accumulate the students' points:

var total = 0;
for (var item in students) {
      if (students.hasOwnProperty(item))
           total += students[item];
}

console.log(`Total points: ${total}`);

In some situations, the object can inherit enumerable properties from Object.prototype. In order to look only at the object's properties, you would use hasOwnProperty() as shown above.

do...while, for, and while loops are not suitable for this example.

The doâ€¦while loop works well when you want to present a menu to the user because this loop is executed at least once and that is what is needed for the menu (present it at least once for the user to make a selection).

The for loop works well in iterating over an array of elements.

The while loop is for the cases in which the condition holds true. It can also be used for infinite loops (while(true)).
