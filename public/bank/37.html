Program with Variables, Data Types, and Functions
SubObjective:

Declare, and use arrays

You have the following HTML page:

<!DOCTYPE html>
<html>
<head>
      <title>Intro to JavaScript</title>
           <meta charset="utf-8" />
</head>
<body>
      <script>
           var arr = ["one", "two", "three", "four"];
           var x = arr.pop();
           var y = arr.shift();
           var z = arr.unshift("five");

           console.log(arr);
      </script>
</body>
</html>

What shows in the console when the page loads?

Array [ "one", "five", "two" ]
Array [ "one", "two", "three" ]
Array [ "five", "two", "three" ]
Array [ "two", "three", "five" ]

Explanation

The following will show in the console:

Array [ "five", "two", "three" ]

The initial declaration of the array starts as follows:

["one", "two", "three", "four"]

Then, the pop operation removes an element from the end of the array. This leaves:

["one", "two", "three"]

The shift operation removes the first element, resulting in:

["two", "three"]

And finally, the unshift operation adds "five" to the start of the array:

["five", "two", "three"]

The console.log(arr) function prints the array in a comma-separated list.
