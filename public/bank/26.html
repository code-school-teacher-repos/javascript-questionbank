The script in the following HTML page prompts the user for values and shows the results in an alert popup window:

<!DOCTYPE html>
<html>
<head>
      <title>Intro to JavaScript</title>
           <meta charset="utf-8" />
</head>
<body>
      <script>
           var input = prompt("Enter a window size between 1 and 50", 25);
           var result = -1;
           input = parseInt(input);
           if (input == null)
                result = 10;
           else if (input == undefined)
                result = 20;
           else {
                for (var i = 0; i < input; i++) {
                     if (i % 2 == 0)
                          result += i;
                     else
                          result += i + 1;
                }
           }
           alert('The "result" is: ' + result);
      </script>
</body>
</html>

What will be the alert if the user enters the value 8?

The "result" is: 39
The "result" is: 10
The "result" is: 31
The "result" is: 20

Explanation

The alert displays the following:

The "result" is: 31

The script starts by prompting the user to enter a value. The second parameter to the prompt is 25, which is the default value that will show up in the textbox. The question states that the user enters 8. That means the value 8 is stored in the variable input.

Any value returned by the prompt is a string (even if the user enters a numeric value). So input now contains "8". The parseInt() function converts the input into an integer. So now input contains 8. Therefore, the check for the input being null or undefined will not be true, and the values 10 and 20 will not display in the alert.

The else clause contains a for loop that that iterates from 0 to 7 (i < input). Every iteration checks whether the value is even using i%2. If it is, then i is added to the result. Otherwise, i + 1 is added to the result. The final outcome is that result contains 31. And that value is placed in the alert.
