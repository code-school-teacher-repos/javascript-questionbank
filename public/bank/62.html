Implement and Analyze Decisions and Loops
SubObjective:

Evaluate expressions that use logical and comparison operators

You have the following declarations:

var x = 7;
var y = 13;
var z = true;
var a = null;
var b = 0;
var c = -1;
var d = " ";

You need to complete the code so the process can start.

Which expressions should you use? To answer, select the appropriate options from the drop-down menus.

Explanation

The code block should be completed as follows:

if ((undefined || {}) && (++x > 7 && y-- <= 13))
startProcess();

With both expressions being true, the startProcess() function will be called.

undefined is not true, but any object is. Since {} is true, (undefined || {}) will evaluate to true.

Incrementing x will give 8, which is greater than 7. Also y, being 13, is less than or equal to 13. Therefore, the expression (++x > 7 && y-- <= 13) is true as well.

The expression (d.trim() || a) is false. Trimming d will result in an empty string, which is false, and a, being null is also false.

The expression (NaN && z) is also false. Although z is true, NaN is false and (false && true) is false.

The expression (c && b) is false. c being -1 is true, but b is false since it is 0.

The expression (z && (++c)) is also false. z is true, but ++c will increment c to 0, which is false.
